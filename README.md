# Zero to Hero

## Setup

* Install [nvm](https://github.com/coreybutler/nvm-windows/releases)
* Run `nvm use` from the project root
* Run `npm i`

## env files

* Copy `.env.template` to `.env`
* Fill in the values with secrets

## Running the server

To run the development server:
```
npm run dev
```

To run the linter:
```
npm run lint
```
